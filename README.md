## 🌟 Sobre o Template

O repositório **UNPCODE** fornece um **ambiente de desenvolvimento pré-configurado** (via Codespaces) altamente focado para as aulas de programação. Ele inclui todos os softwares, bibliotecas e ferramentas necessários para você começar a programar imediatamente, sem a necessidade de instalar nada em seu computador pessoal.

Este foi criado como um template padrão para as disciplinas de **Programação I** do **[Professor Ed](https://edkallenn.github.io/)** do Centro Universitário Unipê (João Pessoa - Paraíba).

## 💻 Instruções do Terminal

**Comandos do terminal**

| Comando | Descrição |
| --- | --- |
| `ls` | Lista os arquivos e diretórios |
| `cd <diretório>` | Muda o diretório atual |
| `cd ..` | Volta um diretório |
| `mkdir <diretório>` | Cria um diretório |
| `code <arquivo>` | Abre o arquivo no VS Code |
| `rm <arquivo>` | Remove o arquivo |
| `clear` | Limpa a tela |

Para compilar e executar um arquivo em C no terminal, use os seguintes comandos:

**Usando o utilitário Make:**
```bash
make <nome_do_arquivo>
```

**Ou alternativamente, invocando o GCC:**
```bash
gcc <nome_do_arquivo>.c -o <nome_do_arquivo>
```

**Para executar o arquivo gerado (binário), no terminal faça:**
```bash
./<nome_do_arquivo>
```

> ⚠️ *Obs*: se a biblioteca `math.h` for usada no seu código, lembre-se de adicionar o parâmetro `-lm` no final do comando de compilação:
> `gcc <nome_do_arquivo>.c -o <nome_do_arquivo> -lm`

## ⌨️ Teclas de atalho

Para agilizar o desenvolvimento, o Codespace conta com as seguintes teclas ativadas:

| Ação | Tecla |
| --- | --- |
| Compilar e executar | `Ctrl-F5` |
| Compilar sem executar | `Ctrl-Shift-B` |
| Depurar | `F5` |
| Indentar automaticamente | `Ctrl-Shift-I` |
